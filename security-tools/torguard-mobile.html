Torguard

Torguard for mobile devices

Go to the [AppStore (for iOS)] or the [Play Store (for Android)] and search for TorGuard app. Proceed to install and open when itâ€™s over

  1. Open your applications menu and tap the newly installed TorGuard icon. Select the VPN server that you wish to connect to by tapping the country name and then tap [Connect].
  2. When prompted, enter username and password and tap the login button.
  3. You should now see a connected status! This means your VPN is active and all traffic is tunneled through the secure server.
  4. To Disconnect just hit the [Disconnect] button at the bottom of the "Connected" window.

Now you're Done!!!
